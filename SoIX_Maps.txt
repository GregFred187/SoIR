# Settler maps (AI found modifiers)

# The numbers work a bit different from those in Rhye's and Fall:
# - values of 3 or less cannot be settled by the AI, period; therefore, use 4+ 
#   if you want at least the possibility
# - there is no colonization so numbers 20-90 never become valid city spots;
#   if you need this kind of effect, look at the original RFC
# - values can be graded; this doesn't have much use in these maps, but can be
#   used to randomize the settler behaviour a bit (see DataLoader.py)
# - technically, each civilization's array can be of any size and the mod will 
#   load even if the they don't match the map size, or even if they are empty,
#   so that you can make/modify the map and worry about AI settlers maps later
# - note that SoI settlers are guided to settle particular plots, unlike in RFC
#   which promotes whole areas; this behaviour depends on map values only, not
#   any special code
#
# 3 = founding disabled for the AI
# 20, 40 = AI will not found unless no better site is available
# 60 = founding possible only with spare settlers
# 90 = as above, but can be become 100+ through randomization
# 100 = default value if no maps are loaded; this means 100% of normal i.e. no effect
